# 情绪化交易压力设计方案

## 设计哲学

> 表层是"帮助交易者获取信息"，底层是"通过信息过载和社交压力诱导情绪化决策"。
> 每一个UI元素都有双重目的：表面功能 + 行为数据采集价值。

---

## 已实现的情绪压力点

### 1. 社交信息栏（SocialBar）— 底部常驻

| 指标 | 表面功能 | 情绪触发机制 |
|------|---------|-------------|
| **L/S Ratio + 5min变化** | 了解市场方向 | 多空比变化带箭头，诱导从众（"64%的人做多，我是不是错了？"） |
| **Win/Loss 分布 + 平均盈亏** | 了解全场表现 | 显示"62%亏损中，平均亏-2.1%"制造焦虑；显示"盈利者平均+3.8%"制造FOMO |
| **Avg Trades对比** | 了解交易频率 | 当你的交易次数低于平均值时，显示黄色"(-X behind)"，诱导过度交易 |
| **晋级线密度** | 了解竞争强度 | 显示"47人在#290-#310争夺"，制造紧迫感 |
| **排名变动** | 了解排名趋势 | 显示"▼3 passed you"红色文字，触发被超越焦虑 |
| **连亏统计** | 了解风险 | 显示"89人连亏3笔+"，暗示"你不是唯一亏的"但同时制造恐慌 |

**关键设计**：随机闪烁某个指标（每8秒），强制吸引注意力到不同的压力源。

### 2. 聊天室（ChatRoom）— 右侧Tab

| 消息类型 | 颜色 | 情绪目的 | 权重 |
|---------|------|---------|------|
| **Brag（炫耀）** | 绿色背景 + 💰 | 嫉妒 → FOMO → 冲动交易 | 高（3x） |
| **Panic（恐慌）** | 红色淡背景 + 😰 | 恐惧传染 → 恐慌性平仓/开仓 | 高（3x） |
| **FOMO** | 普通样式 | 从众压力 → "大家都在做，我也要做" | 中（2x） |
| **Analysis** | 普通样式 | 信息过载 → 分析瘫痪 or 过度自信 | 低（1x） |
| **System Alert** | 金色/红色边框 | 权威信息 → 紧迫感 | 定时 |

**关键设计**：
- Brag 和 Panic 消息权重是普通消息的 3 倍，确保聊天室充满情绪化内容
- 系统广播包含具体数字（"#290-#310有47人"），增加可信度和压力
- 消息不清空（跨场保留），让新进入者看到历史情绪

### 3. 推送通知（CompetitionNotifications）— Toast弹窗

#### 3.1 定时推送（蓝图 Section 2.8）
| 时间点 | 紧急度 | 内容 |
|-------|--------|------|
| T+25% | Info | 排名 + 收益 + 晋级分 + 参与度 |
| T+50% | Info | 排名 + 可提现 + 晋级分差距 + 距50%分成还需X笔 |
| T+75% | Warning | 最后6小时 + 剩余交易次数 |
| T+83% | Warning | 最后4小时 + 倒计时变色提示 |
| T+92% | Critical | 最终结算倒计时 + 红色发光边框 |
| T+96% | Critical | 最后1小时 + 最终冲刺 |

#### 3.2 情绪推送（新增）
| 触发条件 | 内容 | 频率 |
|---------|------|------|
| 被超越 | "过去30分钟有X人超越了你！" | 20-35秒 |
| 交易频率落后 | "全场平均已交易X笔，你才Y笔" | 20-35秒 |
| 积分差距 | "距50%分成还需X积分" | 20-35秒 |
| 可提现焦虑 | "当前可提现X USDT — 保住利润还是冲击更高排名？" | T>70%时 |
| 全场亏损 | "全场62%选手亏损中，平均亏损-2.1%" | 20-35秒 |
| 交易次数告急 | "仅剩X笔交易机会！每一笔都很关键" | 剩余≤10笔时 |

**关键设计**：晋级线附近用户推送频率 ×2。

### 4. 新闻流（NewsFeed + NewsTicker）

#### 4.1 滚动横幅（NewsTicker）
- 高影响力新闻标题更亮、带红色圆点
- BREAKING 新闻带红色闪烁标签
- 持续滚动，制造"信息不断涌入"的感觉

#### 4.2 新闻详情（NewsFeed Tab）
- 高影响力新闻带红色左边框 + "🔴 HIGH IMPACT" 标签
- 中影响力新闻带金色左边框 + "🟡 MEDIUM" 标签
- 情绪标签（↑ Bullish / ↓ Bearish）
- 定期随机"闪烁"某条高影响力新闻

### 5. 状态栏（StatusBar）— 顶部常驻

- 倒计时最后阶段变色：绿 → 黄 → 红 → 闪烁
- 晋级分显示与阈值对比
- 可提现金额始终可见（制造"保住利润"的焦虑）
- 3场周期进度（"第2场/3场"）

### 6. 交易面板（TradingPanel）— 底部

- 持仓权重实时显示（0.2x → 1.3x），诱导"再等等权重就上去了"
- TP/SL 设置，但默认关闭（需要主动开启），增加"裸奔"交易概率
- 持仓PnL实时跳动，绿色/红色强烈对比

## 建议后续增强

### 高优先级
1. **"被超越"实时弹窗**：当有人超越你时，弹出 toast 显示对方用户名和新排名
2. **排行榜动态波动**：排名每30秒随机变化±1-3名，制造持续的排名焦虑
3. **TP/SL 线在K线图上可视化**：绿色/红色虚线，让用户看到"还差多少就触发了"
4. **新闻与价格联动提示**：新闻发布后如果价格剧烈波动，弹出"📰 新闻发布后价格变动 +2.3%"

### 中优先级
5. **聊天室情绪指数**：实时显示聊天室整体情绪（贪婪/恐惧），类似 Fear & Greed Index
6. **交易热力图**：显示"过去5分钟哪个价位开仓最多"
7. **"相似选手"对比**：显示与你排名相近的选手的平均交易频率、持仓时间、盈亏比
8. **倒计时音效**：最后1小时加入滴答声效果

### 低优先级
9. **成就系统**：首次盈利、连赢3笔、权重达到1.0x 等成就弹窗
10. **历史回放**：比赛结束后可回放自己的交易决策时间线
11. **匿名投票**：聊天室内"你现在做多还是做空？"实时投票

## 行为数据采集价值

每个情绪压力点都对应一个可采集的行为信号：

| 压力点 | 可采集信号 | 分析价值 |
|-------|-----------|---------|
| L/S Ratio 变化后 | 用户是否跟随多数方向开仓 | 从众倾向评分 |
| Brag 消息后 | 用户是否在30秒内开仓 | FOMO 敏感度 |
| Panic 消息后 | 用户是否恐慌性平仓 | 恐惧敏感度 |
| 被超越通知后 | 用户是否加大仓位 | 竞争焦虑评分 |
| 新闻发布后 | 用户是否在60秒内交易 | 新闻驱动交易倾向 |
| 倒计时变色后 | 交易频率是否增加 | 时间压力敏感度 |
| 交易次数落后提示后 | 是否立即开仓 | 社交比较敏感度 |
| 可提现金额变化后 | 是否选择保守策略 | 损失厌恶评分 |
