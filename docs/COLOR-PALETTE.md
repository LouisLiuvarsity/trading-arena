# Otter Trader — Color Palette Reference

本文档汇总了 Trading Arena 项目中使用的所有颜色定义，包括 CSS 主题变量、交易专用色彩、组件级硬编码色值，以及动画和特效中使用的颜色。所有主题色均采用 **OKLCH** 色彩空间定义（Tailwind CSS 4 要求），硬编码色值以 **HEX** 或 **RGBA** 格式存在于各组件中。

---

## 1. 设计理念

Otter Trader 采用深色交易终端风格，以 **深蓝黑色** 为底色，**金色（Gold）** 为品牌主色调，搭配 **绿色（Profit）** 和 **红色（Loss）** 作为交易方向指示色。整体视觉参考了 Binance 等专业交易平台的配色体系，同时融入了独特的金色品牌标识。

## 11. 颜色使用规范

### 推荐做法

1. **优先使用 CSS 变量**：通过 Tailwind 类如 `bg-background`、`text-foreground`、`border-border` 引用主题变量，确保全局一致性。
2. **交易色使用语义类**：盈亏显示使用 `.text-profit` / `.text-loss`，品牌强调使用 `.text-gold`。
3. **透明度变体保持一致**：同一语义的透明度应在整个项目中保持统一（如标签背景统一用 12%，悬停用 15%）。

### 需要注意

1. 项目中存在大量**硬编码 HEX 值**（约 50+ 个唯一色值），分布在 70+ 个组件文件中。后续可考虑将高频色值统一提取为 CSS 变量。
2. 部分组件同时使用了 OKLCH 主题变量和 HEX 硬编码，建议逐步迁移到统一的变量体系。
3. 橙色系（`#FF6B35`、`#FF7A00`）目前未纳入主题变量，如需全局调整需逐文件修改。
